# ChatKit UI Embed

## Overview
Drop-in ChatKit UI template for embedding an agent chat experience with hosted tools and document search.

## Prerequisites
- Node.js 18+
- `OPENAI_API_KEY`

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```
