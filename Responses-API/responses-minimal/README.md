# Responses API - Minimal Starter

The simplest possible Responses API example. Start here if you're new.

## Python

```bash
cd python
pip install -r requirements.txt
python main.py
```

## TypeScript

```bash
cd typescript
npm install
npx tsx main.ts
```

## What it does

1. Creates a simple response request
2. Prints the model's output
